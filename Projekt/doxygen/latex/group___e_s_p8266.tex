\doxysubsection{Detailed Description}
E\+S\+P8266 W\+I\+FI library. 


\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{preprocessor}{\#include <\mbox{\hyperlink{_e_s_p8266_8h}{ESP8266.h}}>} }
\end{DoxyCode}


The library can be used to both, connect to a wifi network, and then open T\+CP connections, and send data and receive data.

You can also use it to start your own access point and receive data, acting as a T\+CP server.
